# VERCEL DEPLOYMENT - QUICK START GUIDE

## 🚀 Deploy in 5 Minutes!

### What You Need:
- The HTML file (you already have it)
- A free GitHub account (takes 1 minute to create)
- A free Vercel account (connects to GitHub)

---

## STEP-BY-STEP INSTRUCTIONS

### STEP 1️⃣ : Create GitHub Account (If Needed)
1. Go to → https://github.com/signup
2. Enter email, password, username
3. Verify email
4. Done! ✅

### STEP 2️⃣ : Create & Upload to GitHub Repository
1. Go to → https://github.com/new
2. Repository name: `rama-navami-tracker`
3. Description: "Sri Rama Navami Head Count Tracker"
4. Click **"Create repository"**
5. You'll see a screen with upload options
6. Click **"uploading an existing file"** or **"Add files → Upload files"**
7. Download all these files and upload them:
   ```
   - rama_navami_tracker.html (rename to index.html)
   - vercel.json
   - README.md
   ```
8. Click **"Commit changes"** ✅

### STEP 3️⃣ : Connect to Vercel & Deploy
1. Go to → https://vercel.com
2. Click **"Sign Up"**
3. Click **"Continue with GitHub"**
4. Authorize Vercel
5. Click **"Import Project"**
6. Select your `rama-navami-tracker` repository
7. Click **"Deploy"**
8. Wait 30 seconds... 🎉

### STEP 4️⃣ : Get Your Live URL
- Vercel shows your URL: `https://rama-navami-tracker-[random].vercel.app`
- Copy this URL!

### STEP 5️⃣ : Share in WhatsApp! 🙏
```
🙏 Sri Rama Navami Celebration 🙏

Head Count Form:
https://your-url-here.vercel.app

Please confirm your participation!
```

## ❓ TROUBLESHOOTING

**"I don't see my repository"**
→ Check you're logged into the same GitHub account in Vercel

**"Deployment failed"**
→ Make sure you have `index.html` in your repository (not `rama_navami_tracker.html`)

**"The form doesn't work"**
→ Try a different browser or clear browser cache

**"How do I update the form?"**
→ Edit `index.html` on GitHub → Vercel auto-redeploys in 1 minute
